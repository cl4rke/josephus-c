# Josephus Permutation
The Josephus permutation done with a sequential circular queue in C.

## Input
```
Players count (N): 5
Passes count  (L): 3

Players count (N): 22
Passes count  (L): 5

Players count (N): 5
Passes count  (L): 9

Players count (N): 2
Passes count  (L): 1

Players count (N): 22
Passes count  (L): 2
```

## Output
```
5
18
3
1
14
```
